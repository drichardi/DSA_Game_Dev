Core Concepts: "The Big Three"

Unity is not just a coding tool; it's a "Component-Based" engine. To succeed in this class, you need to understand the relationship between Scenes, GameObjects, and Components.

1. GameObject

The Actor / The Container.
A GameObject is just an empty box. By itself, it can't do anything. It has no shape, no logic, and no physics. It is just a container that holds a position in the world.

2. Component

The Role / The Sticky Note.
Components give GameObjects their power. Want it to render? Add a MeshFilter. Want it to fall? Add a Rigidbody. Want it to move? Add a Script.

3. Scene

The Stage / The World.
The Scene file (.unity) is the container for everything else. It saves the state of all GameObjects and their Components.

ðŸ’¡ The "Cardboard Box" Analogy

Imagine a plain cardboard box. That is a GameObject. It's boring.
If I stick a note on it that says "Gravity", the box falls. If I stick a note on it that says "Shiny", the box reflects light. These notes are Components.
We build games by sticking different notes (Components) onto different boxes (GameObjects).

The Interface Tour

If you are opening Unity for the first time, ensure your layout allows you to see both the Scene View (where you build) and the Game View (what the player sees) at the same time.

  • Hierarchy Window (Left): A list of every GameObject currently in the Scene.
  • Scene View (Center): The 3D construction zone. Fly around using Right-Click + WASD.
  • Inspector Window (Right): Shows the Components attached to the currently selected GameObject.
  • Project Window (Bottom): Your file explorer. Contains all your assets (Textures, Scripts, Models).

Lab: The Solar System

In class, we built a simple Solar System to demonstrate Hierarchy (Parenting). Follow these steps to catch up:

1 Create the Sun

Right-click in the Hierarchy > 3D Object > Sphere.
Name it "Sun".
In the Inspector, set its Position to 0, 0, 0 and Scale to 2, 2, 2.

2 Create the Earth

Create another Sphere. Name it "Earth".
Set its Position to 5, 0, 0 (X = 5).
It should be sitting to the right of the Sun.

3 The Power of Parenting

Try rotating the Sun (press E for Rotate tool). Notice the Earth stays still.
Now, drag "Earth" onto "Sun" in the Hierarchy.
Earth is now a "Child" of Sun. Rotate the Sun again. The Earth should orbit around it!

4 The Moon (Challenge)

Create a smaller Sphere named "Moon".
Make it a child of the Earth.
Goal: Can you make the Sun rotate (carrying Earth), while the Earth rotates (carrying the Moon)?
